OUR PROJECTS

Can Pooper

One of the first Discord bots that was created by a Li Institution member that still exists, Can Pooper is one of the messiest Li Institution codebases that is still considered active and alive. Besides being messy and badly architectured, Can Pooper does not possess any specific purposes and is mainly created for the purpose of amusement. It is written in Python (check out the source here). You can also check out its features by joining the Li Institution Discord server.

Chess Program

A simple chess program written in C++. It is currently not finished, but you can still check it out here. Besides the CMake build scripts and the GLSL shaders, it is written completely in C++ and uses OpenGL for rendering. It uses the GLFW library for window and context creation, and uses GLAD for loading OpenGL functions. Like Can Pooper, it does not possess good software architecturing, but unlike Can Pooper, it has some kind of architecture whereas Can Pooper is just a bunch of program flow control statements.

FHDBot

As the name suggests, this Discord bot is created by Anthony Huang. Like Can Pooper, it currently possesses no specific purpose and is mainly created for amusement purposes. As a result, like Can Pooper, it's code architecture isn't exactly the best and likely has a few hidden bugs throughout the codebase. It is written in JavaScript and uses the Discord.js library for interacting with the Discord API. You can check it out here or by joining the Li Institution Discord Server.
